## Go开发小技巧&易错点100例

各位Gopher大家好！

这个仓库主要是想总结一下自己Coding过程中遇到的问题以及平时读一些博客的所得，因为作为一名Gopher也有了一段时间了，相比Java，有些问题的出现想要利用搜索引擎排查出来可能不是那么的迅速，所以在这里以文章的形式总结出来也方便各位初出茅庐的gopher们能够顺利的解决所遇到的问题，并能够习得一些小技巧。

为什么叫《Go语言开发小技巧&易错点100例》呢，说实话我也不知道能不能写到100例，只能说作为自己的一个小目标吧，先赚它一个亿，哈哈哈，只有目标才能促使自己不断Coding，不断发现和总结问题，相信到最后肯定要多于100个的！
